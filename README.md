# Weather App

## Overview of the Project

Welcome to the Weather App project! This application provides real-time weather updates for any location using data from the OpenWeather API. The project was developed as part of a six-week internship at CodeTech IT Solutions, under the mentorship of Sravani Gouni. This app showcases various aspects of web development, including API integration, responsive design, and dynamic content rendering.

## Technologies Used

- **HTML**: For structuring the content on the web page.
- **CSS**: For styling the application, including layout, colors, and responsiveness.
- **JavaScript**: For adding interactivity and dynamically fetching and displaying weather data.
- **OpenWeather API**: For retrieving current weather information based on user input.

## Features

1. **Real-Time Weather Updates**: 
    - Fetches and displays current weather data including temperature, weather description, humidity, wind speed, and atmospheric pressure.
2. **User-Friendly Interface**: 
    - Clean and intuitive design with responsive layout for various devices.
3. **Search Functionality**: 
    - Users can search for weather updates by entering the name of any location.
4. **Error Handling**: 
    - Provides user feedback when a location is not found.
5. **Dynamic Weather Icons**: 
    - Displays weather-specific icons to visually represent the current weather conditions.

## How to Use

1. **Enter Location**: Type the name of the location you want to check the weather for in the search box.
2. **Get Weather Updates**: Click the search button to fetch and display the current weather data for the entered location.
3. **View Weather Details**: The app will display the temperature, weather description, humidity, wind speed, and pressure for the selected location.

## Project Structure

- **index.html**: The main HTML file that structures the web page.
- **style.css**: The CSS file that styles the application.
- **script.js**: The JavaScript file that handles interactivity and API integration.
